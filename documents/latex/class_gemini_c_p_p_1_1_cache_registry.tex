\doxysubsection{Detailed Description}
Manages a local registry of cached content. 


\begin{DoxyItemize}
\item Allows developers to map friendly names (aliases) to complex server-\/side IDs.
\item The registry is persisted to a JSON file (default\+: "{}gemini\+\_\+caches.\+json"{}). 
\end{DoxyItemize}
