GeminiCPP 1.0.0-alpha
Gemini-cpp is a lightweight C++ client library for the Google Gemini API. It provides a simple, modern interface for sending prompts, handling responses, and working with structured model outputs. Designed with minimal dependencies and straightforward usage in mind, it lets you integrate Gemini into C++ projects without dealing with HTTP boilerplate or complex setup.

115 T parseArg(const nlohmann::json& j, const std::string& name)
116 {
117 if (!j.contains(name))
118 {
119 if constexpr
120 (std::is_default_constructible_v<T>)
121 return T{};
122
123 throw std::runtime_error("Missing argument: " + name);
124 }
125
126 if constexpr (frenum::is_frenum_v<T>)
127 {
128 auto val = frenum::cast<T>(j.at(name).get<std::string>());
129 if(val)
130 return *val;
131
132 throw std::runtime_error("Invalid enum value for: " + name);
133 }
134 else
135 {
136 return j.at(name).get<T>();
