GeminiCPP 1.0.0-alpha
Gemini-cpp is a lightweight C++ client library for the Google Gemini API. It provides a simple, modern interface for sending prompts, handling responses, and working with structured model outputs. Designed with minimal dependencies and straightforward usage in mind, it lets you integrate Gemini into C++ projects without dealing with HTTP boilerplate or complex setup.
