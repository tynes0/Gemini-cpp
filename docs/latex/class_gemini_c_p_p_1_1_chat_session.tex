\doxysubsection{Detailed Description}
Manages a multi-\/turn conversation with the Gemini model. 


\begin{DoxyItemize}
\item This class maintains the history of the conversation, handles function calling loops (auto-\/reply), and manages session-\/specific configurations like safety settings and tools. It is thread-\/safe for modifying settings and sending messages. 
\end{DoxyItemize}
