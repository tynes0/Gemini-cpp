\doxysubsection{Detailed Description}
Represents cached content on the server. 


\begin{DoxyItemize}
\item Caching allows reusing large contexts (like books or codebases) across multiple requests without re-\/uploading or re-\/processing them, saving tokens and time. 
\end{DoxyItemize}
