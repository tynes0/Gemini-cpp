\doxysubsection{Detailed Description}
The main client class for interacting with the Google Gemini API. 


\begin{DoxyItemize}
\item This class acts as the central entry point for all API operations. It manages the API key, network configurations, and provides access to specific API modules (\doxylink{class_gemini_c_p_p_1_1_files}{Files}, \doxylink{class_gemini_c_p_p_1_1_models}{Models}, \doxylink{class_gemini_c_p_p_1_1_tokens}{Tokens}). It also handles the core content generation methods. 
\end{DoxyItemize}
